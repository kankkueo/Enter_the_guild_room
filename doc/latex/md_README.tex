 Enter the Guild \doxylink{classRoom}{Room} is a rogue-\/like dungeon crawler top-\/down shooter written in c++. In the game, the player has to get through a set of randomized rooms with increasingly difficult random monsters with random weapons until they reach the final boss. Defeating the boss ends the game. The game is reset upon player death.\hypertarget{md_README_autotoc_md1}{}\doxysection{\texorpdfstring{Build instructions}{Build instructions}}\label{md_README_autotoc_md1}
\hypertarget{md_README_autotoc_md2}{}\doxysubsubsection{\texorpdfstring{Prerequisites}{Prerequisites}}\label{md_README_autotoc_md2}
Compiling the project requires gcc and make.

The following libraries also have to be installed on your system\+:


\begin{DoxyItemize}
\item sdl2
\item sdl2\+\_\+image
\item sdl2\+\_\+ttf
\item sdl2\+\_\+mixer
\end{DoxyItemize}

These can be installed by running ~\newline
 {\ttfamily sudo apt install libsdl2-\/2.\+0-\/0 libsdl2-\/image-\/2.\+0-\/0 libsdl2-\/ttf-\/2.\+0-\/0 libsd2-\/mixer-\/2.\+0-\/0}\hypertarget{md_README_autotoc_md3}{}\doxysubsubsection{\texorpdfstring{Compiling}{Compiling}}\label{md_README_autotoc_md3}
Run {\ttfamily make} in the project root. ~\newline
 {\ttfamily make run} starts the game.\hypertarget{md_README_autotoc_md4}{}\doxysection{\texorpdfstring{Playing the game}{Playing the game}}\label{md_README_autotoc_md4}
After starting the game, select "{}\+New game"{} from the menu and the game spawns you into a room. Navigate the manu with arrow keys and use ENTER to select.\hypertarget{md_README_autotoc_md5}{}\doxysubsubsection{\texorpdfstring{Controls}{Controls}}\label{md_README_autotoc_md5}

\begin{DoxyItemize}
\item WASD keys\+: Move around
\item Arrow keys\+: Shoot
\item E\+: Interact with items (switch weapons, pick up healing potions, enter the next room)
\item ESC\+: pause the game 
\end{DoxyItemize}
